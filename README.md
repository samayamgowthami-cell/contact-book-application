# Contact Book (Python)

A simple command-line Contact Book application built using Python.  
It allows users to add, view, search, and delete contacts using a dictionary-based storage system.

## Features
- Add new contacts
- View all saved contacts
- Search contact by name
- Delete a contact
- Menu-driven interface

## Technologies Used
- Python

## How to Run
1. Clone this repository
2. Run the file:
   python contact_book.py
